# Applied R Programming

This repository contains a curated collection of applied work in R, developed through BYU Stat 250. It highlights practical experience in data analysis, visualization, and statistical programming, with an emphasis on writing clear, reproducible code.

## Overview

The goal of this repository is to document and demonstrate:
- Data wrangling and cleaning using R
- Exploratory data analysis (EDA)
- Statistical modeling and interpretation
- Data visualization and communication
- Reproducible workflows

Rather than serving as a simple archive of coursework, this repository is organized to showcase projects and analyses that reflect key concepts and practical problem-solving skills.

## Structure

- `projects/` — Selected analyses and larger assignments  
- `labs/` — Shorter exercises and concept-focused work  
- `visuals/` — Images and graphics used across files  

## Tools & Technologies

- R
- RStudio
- tidyverse (dplyr, ggplot2, etc.)
- R Markdown

## Notes

All work is based on coursework from BYU Stat 250 and is presented here in a refined and organized format. Some files may be adapted or cleaned up from their original versions to improve clarity and readability.
